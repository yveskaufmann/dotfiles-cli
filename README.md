# My personal dotfiles

These dotfile are used by among differnt machines:

- Ubuntu 22.04
- Mac OSX
- Windows WSL2 Ubuntu

# Setup

1. Clone the .dotfiles into the home directory:

```sh
cd ~
git clone --recursive https://github.com/yveskaufmann/.dotfiles.git
```

2. Invoke bootstrap to initialize the dotfiles

```sh
.dotfiles/bin/bootstrap
```
